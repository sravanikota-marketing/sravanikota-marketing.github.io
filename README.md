# Sravani Kota Portfolio

A fully static, recruiter-friendly marketing portfolio built with React + Vite for GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## What to update before publishing

- Replace placeholder contact details in `src/data/siteContent.js`.
- Replace placeholder testimonials and product marketing case studies in `src/data/siteContent.js`.
- Confirm the resume files in `public/assets/resume/` are the latest versions.

## GitHub Pages

This repo is configured for a user/organization GitHub Pages site (base path `/`).
If you deploy as a project site, update `base` in `vite.config.js` to `'/<repo-name>/'`.
